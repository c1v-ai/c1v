The architectural reasoning behind Path B (per HANDOFF-2026-04-27 R-V22.B): each agent reads FROM the substrate. Not THROUGH a feeder pipe. The agent-emitter pattern that put structured artifact emission INSIDE an LLM call collapsed under three pressures:
