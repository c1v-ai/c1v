# Plan 03-01: Diagnose Collision Root Cause, Fix Walker, Re-ingest Local + Prod

## Objective

Root cause (VEC-01): Phase A ingest ran with EMBEDDINGS_STUB=1, populating local kb_chunks with 4,990 stub-embedded rows. Phase B ON CONFLICT DO NOTHING cannot replace stub rows with real embeddings since the (kb_source, chunk_hash) key matches. Prod was never touched (0 rows).

Additional bug: walker includes _legacy_2026-04-26 (1,378 chunks) and _dev-runbooks (10 chunks) — these are internal artifacts, not KB content.

## Tasks

### Task 1: Document root cause (VEC-01)
Write DIAGNOSIS.md with evidence from dry-run output and kb-embedder logic.

### Task 2: Fix SKIP_DIR_PATTERNS in ingest-kbs.ts
Add patterns to skip underscore-prefixed internal dirs:
- `/^_legacy_/` — legacy archived content
- `/^_dev-runbooks$/` — developer runbooks, not KB content

### Task 3: Verify dry-run with fix applied
Run `--dry-run` after the fix. Expected: ~3,450 chunks across 10 sources (no _legacy_, no _dev-runbooks). Document chunk counts per source.

### Task 4: Clear local stub rows
```sql
DELETE FROM kb_chunks;
```
Run against local Supabase (postgresql://postgres:postgres@localhost:54322/postgres).
Verify count = 0 after delete.

### Task 5: Re-ingest local with real embeddings
```bash
set -a; source .env.local; set +a
POSTGRES_URL=postgresql://postgres:postgres@localhost:54322/postgres \
  pnpm tsx scripts/ingest-kbs.ts
```
Expected: >3,000 chunks inserted, 0 skipped. Document inserted count.

### Task 6: Verify local (VEC-02 local)
Run postInsertVerify via the ingest script's built-in verify step. Confirm:
- kb_chunks row count > 0
- vector index exists (ivfflat or hnsw)
- spot-checks per source return non-zero counts

### Task 7: Re-ingest prod with real embeddings (VEC-02/VEC-03)
```bash

POSTGRES_URL=$PROD_POSTGRES_URL \
  pnpm tsx scripts/ingest-kbs.ts
```
Prod is empty — no delete needed. Expected: all chunks inserted fresh.

### Task 8: Verify prod (VEC-03)
Run a targeted count query against prod to confirm kb_chunks has rows with non-null embeddings.
