# Crawley Ch 12 — Applying Creativity to Generating a Concept

## Unstructured creativity (§12.2, line 5577)

Brainstorming, free association, "aha moments," de Bono's random-word technique. Rooted in unconscious processing.

## Structured creativity (§12.2, line 5593)

- **Component recombination** — choose one option per decomposed function (the morphological matrix mechanism).
- **Completeness frameworks** — energy-form list (linear-kinetic / rotational-kinetic / potential / chemical / etc.).
- **de Bono's Six Hats** — managing, information, emotions, discernment, optimistic response, creativity.
- **TRIZ** — Altshuller's 40 inventive principles from 40,000 patent analyses.

## Four-step concept framework (line 5655)

1. **Develop the Concepts** — start from system problem statement, analyze for SN operands/processes, specialize, check against SPS.
2. **Expand the Concepts and Develop Concept Fragments** — unpack rich multifunctional concepts into principal internal functions, create fragments for each.
3. **Evolve and Refine Integrated Concepts** — systematically search concept-fragment space, combine combinatorially with constraints.
4. **Select a Few Integrated Concepts for Further Development** — backwards consideration (fit to goals), forward consideration (likely to make good architecture).

## Concept-ballooning heuristic (Figure 12.10, line 5865)

`1 → N → (3-4)N → (3-4)N! → 10–20 qualitative down-select → 5–7 quantitative down-select → 2–3 final`

The decision network consumes options at the 5–7 quantitative stage; anything upstream is pre-decision-net filtering.

## Pugh matrix (Table 12.1, line 5829)

5-level rating scale (`++, +, o, -, --`) against a reference concept. Rate each candidate against each goal, tagged *critically / importantly / desirably*. Reference = `o` row. Identify architecturally-discriminating goals (those where candidates spread most).

## Worked example: hybrid vehicle (§§12.3–12.5)

7 concept fragments (motor-start-stop, regenerative-braking, power-boost, load-level-increase, electric-driving, external-battery-charging, gliding) × 4 powertrain pathways (steam, ICE, battery, fuel-cell) → 7 integrated concepts (conventional-ICE, micro-HEV, mild-HEV, full-HEV, PHEV, BEV, FCEV).

## KB-4 operational use

The morphological matrix + Pugh down-select produces the **option space per decision node**. The decision-network scoring (KB-4 canonical math) then consumes these options and evaluates utility + sensitivity + Pareto position.
