## Critical Bug: 504 Gateway Timeout in Production

**Error:** `POST /api/chat/projects/103` → 504 Gateway Timeout
**Environment:** Vercel Pro (300s max timeout)

### Root Cause (verified from Vercel logs)

Steps 3-6 graph nodes run **sequentially** and exceed 300s total:
- Extraction: ~2 min
- FFBD: ~2 min
- Decision Matrix: ~26s
- QFD: starts, then **timeout at 300s** — never reaches Interfaces

Total pipeline: ~5 min sequential, Vercel kills at 300s.

### The Fix

**Run FFBD, Decision Matrix, QFD, and Interfaces in parallel.** They're independent — all read from the same extraction result. This cuts ~4.5 min sequential to ~2 min (longest single call).
